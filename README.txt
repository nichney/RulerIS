RulerIS - программа, которая угадывает правителя
России, начиная от Рюрика и заканчивая текщим правителем. Процесс
отгадывания происходит с помощью ответов пользователем на вопросы с возможными вариантами ответа
"Да" и "Нет". В правом верхнем углу доступна кнопка
"Сначала", которая включает на экране первый вопрос системы.

Узнать последнюю версию программы можно по ссылке https://github.com/nichney/RulerIS
